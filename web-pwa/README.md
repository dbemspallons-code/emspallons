# Car Management - Web PWA

Placeholders Firebase disponibles dans `src/firebase/firebaseConfig.js`.
- Installer : `npm install`
- Lancer : `npm run dev`
- PWA : ouvrez avec HTTPS ou `localhost` pour tester l'installation.
- WhatsApp (CallMeBot) : modifiez `src/services/whatsappService.js` pour ajouter votre clé API si nécessaire.
- Export CSV : bouton 'Exporter CSV' sur la page principale.
